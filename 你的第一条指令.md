### 命令
使用如下代码开始创建并注册一个基本的指令
```csharp
public static void yourCommandName(OS os, string[] args)
{
    os.write("Arguments passed in: " + string.Join(" ", args));
}
```
```csharp
Pathfinder.Command.CommandManager.RegisterCommand("commandNameInHacknet", yourCommandName);
```
请保证注册位置与对应的方法名相同 ，并将 `commandNameInHacknet` 替换为你想要的指令在Hacknet中的名称。

这条指令的效果是，当玩家输入 `commandNameInHacknet` 时，它将会输出 `Arguments passed in: ` 后面跟着玩家输入的所有内容。

如：>>> commandNameInHacknet hello world maybe sth else?

输出：Arguments passed in: hello world maybe sth else?

指令一般情况下在Extension中对玩家用处不算大，但是你可以写一些使用的debug指令，辅助你的Extension测试。

下面我将附上一张原版已经有的常用debug指令列表，供参考。（使用  -enabledebug -enablefc 带参数启动分别将启用debug指令和强制完成任务）

·`fh` 快速破解当前计算机
·`dscan` 扫描指定compID的节点
·`ra` 显示所有节点
·`reloadext` 从扩展重新加载和更新所有节点。
·`printflags` 将获取的标记打印到终端中。
·`runhackscript[HACKER_SCRIPT_PATH]` 启动黑客脚本并打印终端中的任何错误。
·`loadactions/loadmissions path/to/file.xml` 加载指定的action/mission文件。

Command指令上本质就是一个无内存占用的EXE而且被阉割掉了Update函数和draw函数，所以只能通过命令行来实现一些功能。

例如使用如下指令实现快速打开端口、关闭端口、解决防火墙、直接解决过载代理、添加\移除flag：
```csharp
    [Pathfinder.Meta.Load.Command("addFlags")]//添加flag
    public static void addFlags(OS os, string[] args)
    {
        if (args.Length < 2)
        {
            os.write("Usage: addFlags <flagName>");
            return;
        }
        os.Flags.AddFlag(args[1]);
        os.write($"Added flag: {args[1]}");
        os.write("Current flags:");
        foreach (string f in os.Flags.Flags)
        {
            os.write($"- {f}");
        }
    }

    [Pathfinder.Meta.Load.Command("rmFlags")]//删除flag
    public static void rmFlags(OS os, string[] args)
    {
        if (args.Length < 2)
        {
            os.write("Usage: rmFlags <flagName>");
            return;
        }
        os.Flags.RemoveFlag(args[1]);
        os.write($"Removed flag: {args[1]}");
        os.write("Current flags:");
        foreach (string f in os.Flags.Flags)
        {
            os.write($"- {f}");
        }
    }


    [Pathfinder.Meta.Load.Command("openport")]//开启端口 +端口号
    public static void openport(OS os, string[] args)
    {
        if (args.Length < 2)
        {
            os.write("Usage: openport <portNumber>");
            return;
        }
        if (os.connectedComp != null)
        {
            if (int.TryParse(args[1], out int portNum))
            {
                os.connectedComp.openPort(portNum, os.connectedComp.ip);
                os.write($"Opened port: {portNum}");
            }
            else
            {
                os.write($"Invalid port number: {args[1]}");
            }
        }
        else
        {
            os.write("Not connected to any computer");
        }
    }

    [Pathfinder.Meta.Load.Command("closeport")] //关闭端口 +端口号
    public static void closeport(OS os, string[] args)
    {
        if (args.Length < 2)
        {
            os.write("Usage: closeport <portNumber>");
            return;
        }
        if (os.connectedComp != null)
        {
            if (int.TryParse(args[1], out int portNum))
            {
                os.connectedComp.closePort(portNum, os.connectedComp.ip);
                os.write($"Closed port: {portNum}");
            }
            else
            {
                os.write($"Invalid port number: {args[1]}");
            }
        }
        else
        {
            os.write("Not connected to any computer");
        }
    }

    [Pathfinder.Meta.Load.Command("firewallkill")] //关闭防火墙
    public static void firewallkill(OS os, string[] args)
    {
        if (os.connectedComp != null && os.connectedComp.firewall != null)
        {
            os.connectedComp.firewall.solved = true;
            os.write("Firewall solved");
        }
        else
        {
            os.write("No firewall to solve");
        }
    }

    [Pathfinder.Meta.Load.Command("overloadproxy")] //关闭代理
    public static void overloadproxy(OS os, string[] args)
    {
        if (os.connectedComp != null)
        {
            os.connectedComp.proxyActive = false;
            os.write("Proxy overloaded");
        }
        else
        {
            os.write("Not connected to any computer");
        }
    }
```

其实，command还有些特殊的用法。比如你可以注册一个用于触发特殊效果的command，比如：forkbomb。
这就要配合Extension的剧情演出和一些特殊要求了。
